A fast, lightweight and multilingual Coming Soon & Maintenance Mode plugin for WordPress.

== Description ==

GMS - Coming Soon lets you activate a simple, full-screen Coming Soon page with a custom headline, text and background image. It supports German, Spanish and English based on the site's language and falls back to English for all other locales.
